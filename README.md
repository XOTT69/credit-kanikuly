# credit-kanikuly

Операційний сайт для роботи з військовослужбовцями  
Оформлення кредитних канікул відповідно до чинного законодавства України.

## Структура проекту

- `src/pages/OperatorWorkflow.jsx` — головний компонент із покроковим інтерфейсом для операторів
- `src/components/` — папка із допоміжними компонентами:
  - CreditLimitChecker.jsx
  - DocumentRequirements.jsx
  - MessageTemplates.jsx
  - ApplicationForm.jsx
  - LawReference.jsx

## Запуск проекту локально

1. Клонувати репозиторій:
git clone https://github.com/XOTT69/credit-kanikuly.git
cd credit-kanikuly

2. Встановити залежності:
npm install

3. Запустити проєкт:

npm start

4. Відкрити у браузері:

http://localhost:3000

## Як працювати

- Вибрати тип клієнта
- Крок за кроком виконувати інструкції оператора
- Заповнити форму заявки та додати документи
- Копіювати готові тексти для спілкування з клієнтом
