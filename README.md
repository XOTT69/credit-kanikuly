# credit-kanikuly

Операційний сайт для роботи з військовослужбовцями, оформлення кредитних канікул

## Структура проєкту

- src/pages/OperatorWorkflow.jsx — головний сценарій для оператора
- src/components/CreditLimitChecker.jsx — перевірка кредитного ліміту
- src/components/DocumentRequirements.jsx — модуль для документів
- src/components/MessageTemplates.jsx — готові тексти для спілкування
- src/components/ApplicationForm.jsx — модуль заявки
- src/components/LawReference.jsx — модуль розʼяснень по закону

## Запуск проєкту

1. Клонувати репозиторій
2. Встановити залежності: `npm install`
3. Запустити локально: `npm start`
